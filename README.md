## 🚀 StudyHub.codes - Hasičský vzdělávací portál

Neoficiální studentský projekt zaměřený na podporu studia požární ochrany na OSŠP Kolín.

Tento portál slouží jako interaktivní rozcestník a úložiště studijních materiálů pro studenty. Cílem je zpřehlednit přípravu na školní výuku, procvičování i praktickou část oboru v moderním a rychlém webovém rozhraní.

## 🔗 Odkazy

Živý web: https://studyhub.codes

Status: Ve vývoji

## 🛠️ Funkce webu

Interaktivní materiály:
Přehledně řazené otázky podle předmětů, vyhledávání v obsahu a modal upozornění u položek, kde zatím není přiložené PDF.

Smart vyhledávání:
Rychlé hledání ve studijních otázkách napříč sekcí materiálů.

Procvičování:
Kvíz po tématech a zkouška nanečisto v demonstračním režimu.

Technické kalkulačky:
Převodník tlaku/průtoku a kalkulačka výdrže vzduchu v dýchací láhvi.

Dark/Light mode:
Responzivní design s přepínáním světlého a tmavého režimu.

Praktická příprava:
Sekce praxe, sporty a samostatná podstránka Mini TFA.

## 💻 Použité technologie

HTML5 / CSS3:
Statický web s vlastním UI a responzivním rozložením.

JavaScript (Vanilla):
Logika vyhledávání, modal oken, navigace a interaktivních nástrojů.

GitHub Pages:
Hosting a publikace webu na vlastní doméně.

Další knihovny:
Font Awesome (ikony), Google Fonts (Montserrat).

## 👨‍💻 O projektu

Projekt vznikl jako iniciativa studentů s cílem vytvořit přehledné a moderní prostředí pro studium odborných předmětů v oblasti požární ochrany.

Upozornění:
Tento web není oficiální stránkou OSŠP Kolín a slouží výhradně ke studijním účelům.
